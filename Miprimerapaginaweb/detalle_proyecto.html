ToDo App

Descripción

Desarrollé una aplicación web completa de lista de tareas utilizando Django para el backend y su sistema de templates para el frontend. La aplicación permitirá a los usuarios crear, visualizar, editar y eliminar tareas.

Herramientas y Tecnologías:

  • Backend: Django como framework principal.
  • Base de Datos: SQLite para desarrollo y pruebas (incluida por defecto en Django).
  • Frontend: Django Templates para el renderizado de la interfaz de usuario.
  • Autenticación: Django's auth module para el registro y autenticación de usuarios.
  • Estilo de la Interfaz: CSS básico para un diseño responsivo y atractivo.
